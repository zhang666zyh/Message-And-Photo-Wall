# 开发文档

## 创建 vue3 项目

```shell
mkdir MessageWall
cd MessageWall
vue create web  # 选默认的vue3配置
```

## 生成目录结构

```shell
npm install mddir -g
mddir  # 在directoryList.md中
```

## 装一些依赖

```shell
npm install less less-loader --save
npm install axios --save
npm install vue-axios --save
```

- package.json 添加

```json
"vuex": "^4.0.2"
```
